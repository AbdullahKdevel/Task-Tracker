Help & Guide
Getting Started
➕ Adding a task
Tap the amber + button in the bottom-right corner. Give your task a title, then optionally add notes, a status, priority, due date, tags, time estimate, and recurrence — all in the same form.
✅ Completing a task
Tap the checkbox on the left of any task to mark it done. Tapping it again un-marks it. Completing a task can trigger a small confetti celebration (toggle in Settings).
📂 Expanding a task
Tap anywhere on a task's title or notes to expand it. Inside you'll find a progress slider, a time tracker, subtasks, custom fields, and quick action buttons.
Organising Tasks
🏷️ Status & tabs
Every task has a status: To Do, Ongoing, Review, Done, Blocked, or Waiting. Use the tabs at the top to filter by status, or tap "Next Status" inside an expanded task to cycle through statuses quickly.
🚦 Priority
Set a priority (None, Low, Medium, High) when creating or editing a task. High-priority tasks are tracked in the stats strip at the top.
🔖 Tags
Add freeform tags to any task (type a word, press Enter). Tap the filter icon at the top, then tap a tag chip to show only tasks with that tag. Tap multiple tags to combine filters.
🔍 Search & filter
Tap the filter icon (funnel) in the top bar to reveal a search box, priority filters, and tag filters. Search matches both the title and notes.
↕️ Sorting & views
Use the sort dropdown above your task list to order by date created, due date, priority, progress, alphabetically, or status. Switch between List view and Board (kanban-style columns) using the toggle next to it.
Tracking Progress
📊 Progress slider
Inside an expanded task, drag the slider to set a completion percentage. This shows as a chip on the task card.
⏱️ Time tracker
Tap the play button in an expanded task to start a timer — it counts up in real time. Tap pause to stop and save the elapsed time, or tap the reset icon to clear it.
📋 Subtasks
Inside an expanded task, type into the "Add subtask…" field and tap Add. Check items off individually — your progress (e.g. "2/4 subtasks") shows on the card.
🔁 Recurrence
When creating a task, choose Daily, Weekly, Every 2 weeks, or Monthly to mark it as a repeating task. A recurrence chip appears on the card as a reminder.
Custom Fields
🧩 Creating your own fields
Tap the grid icon in the top bar to open the Field Manager. Add a field name and pick a type (Text, Number, or URL). It'll then appear as an extra input on every task's edit form, and as a chip inside expanded tasks when filled in.
🗑️ Removing a field
In the Field Manager, tap the × next to any field to delete it. This only removes the field definition — existing task data isn't deleted from Firestore, it just stops being displayed.
Account & Data
👤 Signing in
TaskFlow supports Email/Password, Google, GitHub, and Guest mode. Guest mode stores your data on this device's account only — you can add an email & password later from Settings to make it permanent and accessible anywhere.
☁️ Syncing
All tasks, fields, and settings are stored in the cloud (Firestore) and sync in real time. The small dot next to the icons in the top bar pulses amber while saving, and turns solid green once synced.
📤 Export & reset
In Settings → Data, you can export all your tasks and custom fields as a JSON file for backup, or clear everything to start fresh.
Customising the App
⚙️ Display settings
In Settings, toggle whether completed tasks are shown, whether the stats strip is visible, and whether cards use a more compact layout.
🎉 Confetti
Turn the completion celebration on or off — purely for fun, no functional effect.
